# UnityVR
 GoogleVR Plugin Practice

***Unity版本2019.3.9.f1***

**版本問題:**
* using UnityEngine.UI找不到組件
> 解決方法:
> 將Edit->Preferences->ExternalTools內的"Generate all .csproj files."打勾  
> 並刪除檔案  
>>  "Assembly-CSharp.csproj"  
>>  "Assembly-CSharp-Editor.csproj"  
>>  ".sln檔"  

> 刪除後重開編輯器就可以恢復正常
